# ReactNativeChatApp
    -> For Windows
        npx react-native run-android
    
        in case debug console closed
        npx react-native start

    -> For Linux
    -> new terminal and run "npx react-native start"
    -> new terminal and run "npx react-native run-android"

    -> npx react-native link
